## Python QR Code Generator
(My first project on github :))

Simple python script where you can generate a customized qr code with any URL, change their fill and background color and change the qr code's file name (how it's going to be saved in your computer)

## Features:
- Generate a QR code from any website URL.
- Change the background and fill color (hex code or name) 

## Required Module:
Required module: pip install qrcode[pil]

## How to use it
1- Clone or download this repository
2- Download the required module
3- Run your script
4- Provide website URL, colors and file's name
5- Your generated QR code will be saved in the same folder (directory) where your Python script is located and running.
